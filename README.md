## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/username/sway-rice.git
cd sway-rice
```

Copy the configuration:

```bash
cp -r .config/* ~/.config/
```

Restart Sway:

```bash
swaymsg reload
```

Enjoy the rice before I abandon it. AGAIN. (Or maybe neh, i would never know)
